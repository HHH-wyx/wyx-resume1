# 个人网站照片与内容完善指南

## 一、照片完善建议

### 1. 需要准备的照片

| 照片类型 | 尺寸建议 | 格式 | 用途 | 示例 |
|---------|---------|------|------|------|
| 证件照 | 300x300px | JPG/PNG | 个人头像 | 高清商务照，背景纯色（白/蓝） |
| 项目照片 | 800x600px | JPG/PNG | 项目作品集展示 | 数据可视化图表、项目界面截图等 |
| 实践照片 | 1200x800px | JPG/PNG | 工作经历或项目经历 | 实地调研、团队合作照片等 |

### 2. 照片存放位置

将照片存放在 `public` 目录下，建议创建子目录分类管理：

```
public/
├── images/
│   ├── profile.jpg         # 证件照
│   ├── projects/           # 项目照片
│   │   ├── project1.jpg
│   │   ├── project2.jpg
│   │   └── project3.jpg
│   └── practice/           # 实践照片
│       └── visit.jpg
```

### 3. 照片使用方法

在 `app/[lang]/page.tsx` 文件中使用 `Image` 组件引用照片：

```typescript
<Image 
  src="/images/profile.jpg" 
  alt="Profile" 
  width={200} 
  height={200} 
  className="rounded-full object-cover" 
/>
```

### 4. 免费图库资源

如果没有合适的照片，可以使用以下免费商用图库：
- Unsplash（搜索"data visualization"）
- Pexels
- Freepik
- Pixabay

## 二、内容完善建议

### 1. 内容结构优化

| 模块 | 内容建议 |
|------|----------|
| 关于我 | 添加个人宣言、职业目标、性格特点 |
| 教育背景 | 添加具体课程、GPA、排名等 |
| 工作经历 | 添加具体职责、成就数据、技术栈使用情况 |
| 核心技能 | 添加熟练度星级、技能应用案例 |
| 项目作品集 | 每个项目添加：项目背景、解决方案、核心成果、技术栈、链接 |
| 联系方式 | 确认所有联系方式有效 |

### 2. 各模块内容示例

#### 关于我
```
<p className="text-lg mb-4">
  我是一名智能经济专业的学生，具有扎实的数据分析能力和全栈开发技能。
  致力于将经济理论与技术实践相结合，为企业和社会创造价值。
  我热爱学习新技术，善于解决复杂问题，具有良好的团队合作精神和沟通能力。
  我的职业目标是成为一名优秀的数据分析师，帮助企业做出数据驱动的决策。
</p>
```

#### 项目作品集
```
<div className="p-6">
  <h3 className="text-xl font-semibold mb-2">AI推荐算法数据标注与分析</h3>
  <p className="text-gray-600 dark:text-gray-400 mb-4">
    <strong>项目背景：</strong>某电商平台需要优化推荐算法，提升用户转化率。<br />

    <strong>核心成果：</strong>标注准确率达到96%，提升算法效率30%，最终用户转化率提升15%。<br />
    <strong>技术栈：</strong>Python, Pandas, Scikit-learn, Excel
  </p>

```

### 3. 内容语言风格

- **简洁明了**：使用简短的句子和段落
- **数据驱动**：使用具体数字和案例
- **专业术语**：适当使用专业术语，但保持可读性
- **一致风格**：中英文版本保持一致的内容结构

### 4. SEO优化

- 在页面标题和描述中包含关键词
- 使用语义化HTML标签
- 添加合适的alt文本到图片
- 确保网站加载速度快

## 三、技术实现步骤

### 1. 更新个人照片

1. 将照片上传到 `public/images` 目录
2. 在 `app/[lang]/page.tsx` 文件中找到头像部分
3. 更新 `Image` 组件的 `src` 属性

### 2. 更新项目内容

1. 打开 `app/[lang]/page.tsx` 文件
2. 找到对应的模块（如项目作品集）
3. 更新HTML内容
4. 保存文件，开发服务器会自动刷新

### 3. 更新翻译内容

1. 打开 `public/locales.ts` 文件
2. 更新对应的翻译文本
3. 保存文件，开发服务器会自动刷新

### 4. 添加新的模块

1. 在 `app/[lang]/page.tsx` 文件中添加新的HTML结构
2. 添加对应的CSS样式
3. 更新 `locales.ts` 文件中的翻译

## 四、最佳实践

1. **真实性**：所有内容必须真实可靠
2. **定期更新**：每3-6个月更新一次网站内容
3. **隐私保护**：敏感信息（如完整电话）使用脱敏处理
4. **响应式设计**：确保在不同设备上都有良好的显示效果
5. **性能优化**：压缩图片，减少不必要的脚本

## 五、预览和部署

1. **本地预览**：访问 http://localhost:3000/zh 查看中文版本，http://localhost:3000/en 查看英文版本
2. **部署到Vercel**：
   - 将代码推送到GitHub仓库
   - 在Vercel上导入GitHub仓库
   - 配置自定义域名（如wyxhhh.com）
   - 等待自动部署完成

## 六、后续扩展功能

1. **在线简历下载**：使用html2canvas和jspdf生成PDF简历
2. **滚动动画**：集成AOS库添加滚动动画效果
3. **联系表单**：使用Nodemailer实现表单提交功能
4. **博客功能**：添加博客模块展示个人文章
5. **访问统计**：集成Google Analytics或百度统计

通过以上步骤，你可以轻松完善个人网站的照片和内容，打造一个专业、生动的个人履历网站。
